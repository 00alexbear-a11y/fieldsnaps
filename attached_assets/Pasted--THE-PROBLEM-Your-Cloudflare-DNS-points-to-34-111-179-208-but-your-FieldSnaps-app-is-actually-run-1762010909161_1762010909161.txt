Step 1: Deploy Your Replit App
In your Replit workspace, click the "Deploy" button (top right)
Choose "Production Deployment"
Wait for deployment to complete
Replit will give you a deployment URL like:
https://fieldsnaps-production.replit.app or similar
Step 2: Add Custom Domain in Replit
In Replit, go to Deployments (left sidebar)
Click on your active deployment
Scroll down to "Custom Domains"
Click "Add Custom Domain"
Enter: fieldsnaps.com
Replit will show you DNS instructions - write these down!
It will likely say something like:

Add a CNAME record:
Name: fieldsnaps.com
Value: [some-replit-url].replit.app
OR

Add an A record:
Name: fieldsnaps.com
Value: [replit-ip-address]
Step 3: Update Cloudflare DNS
Based on what Replit tells you:

If Replit gives you a CNAME:
Go to Cloudflare DNS
Delete the current A record pointing to 34.111.179.208
Add a CNAME record:
Name: @ (or fieldsnaps.com)
Target: [the-url-replit-gave-you].replit.app
Proxy status: üü† Orange cloud (Proxied)
If Replit gives you an IP address:
Go to Cloudflare DNS
Edit the A record
Change the IP from 34.111.179.208 to [Replit's IP]
Keep Proxy status: üü† Orange cloud (Proxied)
Step 4: Configure Cloudflare SSL
In Cloudflare, go to SSL/TLS
Set encryption mode to "Full (strict)"

Wait 5-10 minutes for changes to propagate
